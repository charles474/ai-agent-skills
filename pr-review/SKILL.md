## What I do

You are in code review mode.

Focus on:

- Code quality and best practices
- Potential bugs and edge cases
- Performance implications
- Accessibility issues that may be introduced
- Security considerations

Provide constructive feedback and suggest fixes (with examples of how to fix issues found) without making direct changes.

The files this applies to are JavaScript, TypeScript, JSX, and TSX files.

## When to use me

Use this when you are performing a code review on any staged/ unstaged work (this includes changed files in the local branch).
